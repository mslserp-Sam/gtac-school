# Images Directory

Place your logo and other images here.

## Logo File
- **Location**: `public/images/logo.png`
- **Recommended size**: 200x50px or similar aspect ratio
- **Format**: PNG (with transparent background) or SVG

## How to Add Your Logo:

1. Place your logo file as:
   - `public/images/logo.png` (recommended)
   - OR `public/img/logo.png`

2. Supported formats:
   - PNG (with transparency recommended)
   - SVG (scalable, best quality)
   - JPG (if no transparency needed)

3. The logo will automatically display in:
   - Main website header
   - Admin panel header
   - Footer (can be added if needed)

## Alternative Logo Locations:
- `public/images/logo.svg` - SVG format
- `public/images/logo.jpg` - JPEG format
- `public/favicon.ico` - Browser favicon (already exists)

## Example:
```
public/
├── images/
│   ├── logo.png        ← Your main logo
│   ├── logo.svg        ← SVG version (optional)
│   └── favicon.png     ← For browser tab
└── favicon.ico         ← Browser icon
```
